## HISDF with ONNX Runtime in Python
```bash
# Run with video file
python demo_hisdf_onnx_34.py -v video.mp4

# Run with webcam
python demo_hisdf_onnx_34.py -v 0

# Run with image directory
python demo_hisdf_onnx_34.py -i <images_dir>
```
